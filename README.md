# Setup

1. **Install GNU Stow**

   ```sh
   sudo apt install stow
   ```

2. **Install Tmux Plugin Manager**

   ```sh
   git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
   ```

3. **Ensure Tmux version is 3.3a or higher**

   ```sh
   tmux -V
   ```

4. **Run Stow**
   ```sh
   stow .
   ```
